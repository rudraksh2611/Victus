Victus Voice Assistant — after setup
=====================================

What was installed
------------------
• VictusMorningBriefing.exe — main program (double-click to run).
• launch_exe_at_logon.cmd — used when "run at sign-in" is enabled in the installer (Task Scheduler).
• During setup you can choose to start Victus automatically when you sign in to Windows (recommended).

First run
---------
The first time you run the app, a short setup window asks for your name, city, voice, and other options. You do not need to edit any files by hand.

Change settings later
---------------------
• Start Menu → Victus Voice Assistant → Change settings  
  or run: VictusMorningBriefing.exe --setup

Support
-------
See TROUBLESHOOTING.txt in the project repository, or the log at:
